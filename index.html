Deploying API server

1. Register account

2. Install Heroku Command Line Interface (CLI)

3. Click on button "Create new app"

4. Connect GitHub repository

  • 5. Set port with process environment vars
  • 6. Manage project dependencies
  • 7. Or set NPM_CONFIG_PRODUCTION to false
  • 8. Enable auto deploys
  • 9. Select branch and press Deploy
  • 10. Done
